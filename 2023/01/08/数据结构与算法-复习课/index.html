数据结构与算法 复习课

本文最后更新于：2023年1月8日 下午

Chapter 1

数据结构

定义

A data structure is a data object together with the relationships among the data members that compose the object.
$$
Data\ Structure = D + R
$$
D is a data object, R is a limited set of relationships of the data members in D.

分类

线性、非线性

逻辑结构、物理结构

算法

定义

an operation sequence of solving a problem

性质

  1. Input specified
  2. Output specified
  3. Definiteness
  4. Effectiveness
  5. Finiteness

排序

掌握所有排序的算法流程、复杂度

选择排序、插入排序、冒泡排序、秩排序、堆排序、希尔排序、快速排序

关注秩排序

数学和证明方法

如数学归纳法、反证法，一般不作为重点

递归要求掌握，重点掌握物理过程，逻辑过程也要求掌握（？）

四个算法思想

递归（重点）、分治、贪心、动态规划（不要求掌握），越靠前的要求越高

空间复杂度、时间复杂度概念

  • Space complexity: the amount of memory a program needs to run to completion
  • Time complexity: the amount of time a program needs to run to completion

例子：顺序查找

经典算法，要求掌握

渐进记号$O,\Theta$
