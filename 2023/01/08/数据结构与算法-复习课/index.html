数据结构与算法 复习课

本文最后更新于：2023年1月8日 下午

Chapter 1

数据结构

定义

A data structure is a data object together with the relationships among the data members that compose the object.
$$
Data\ Structure = D + R
$$
D is a data object, R is a limited set of relationships of the data members in D.

分类

线性、非线性

逻辑结构、物理结构

算法

定义

an operation sequence of solving a problem

性质

  1. Input specified
  2. Output specified
  3. Definiteness
  4. Effectiveness
  5. Finiteness

排序

掌握所有排序的算法流程、复杂度

选择排序、插入排序、冒泡排序、秩排序、基数排序（不用掌握代码，掌握算法流程）、堆排序、希尔排序、快速排序

关注秩排序

数学和证明方法

如数学归纳法、反证法，一般不作为重点

递归要求掌握，重点掌握物理过程，逻辑过程也要求掌握（？）

四个算法思想

递归（重点）、分治、贪心、动态规划（不要求掌握），越靠前的要求越高

Chapter 2

空间复杂度、时间复杂度概念

  • Space complexity: the amount of memory a program needs to run to completion
  • Time complexity: the amount of time a program needs to run to completion

例子：顺序查找

经典算法，要求掌握

渐进记号$O,\Theta$

另一个例子：折半查找

辗转相除法（复杂度不要求）

Chapter 3 线性表

考点

数组、单链表、栈、队列——必考

单链表

注意区分带表头节点和无表头结点的单链表

双链表一般不考

Cursor implementation of Linked Lists

要掌握

Chapter 4 栈和队列

清晰地了解实现以及复杂度，重点在应用，可参考课件例题

Chapter 5 树

二叉树

掌握基本概念

如：度、深度、高度

基本性质

要求清晰，可能考计算

表示方式

数组表示法、连接表示法，与堆等紧密相关

会考读代码、写代码

遍历（先序、中序、后序、层次）——重点

广义表不考，但其他的建二叉树方法都可能靠

字符串（如果考，是送分（？））

树

树的存储方式

  • 广义表不要求
  • 双亲表示法可能在并查集里考察
  • 左子女右兄弟是重点

遍历依然是重点

树、二叉树、森林、图的遍历

线索化二叉树

要求，常考

霍夫曼树

可能考（小概率），考法固定

Chapter 6 搜索树

AVL树、B-树常考

各个算法（操作）复杂度也要掌握

B-树代码不要求掌握，流程掌握，会考画B-树
